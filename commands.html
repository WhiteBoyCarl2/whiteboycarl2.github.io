Introduction

NovaBot is the perfect discord moderation bot for your server. With a great amount, and cool commands. NovaBot will have you covered from Moderation, Utility, and Fun.

About this Bot

This bot is not only made just for your server. But also so you can have fun with it. We supply a wide amount of commands. Ranging from Moderation to Fun. Which is why people love our bot.

Technology

This bot uses the discord.js library, and is always a good choice.

The bot not only uses discord.js, but we also use the MongoDB database.

Getting Started

The bot is fairly easy to setup, all you do is add it, and we do the rest.

Important: You must agree with our Terms of Service, and Privacy Policy..

Adding the bot

To add the bot to your server click here.

This does needs you to either be the owner of the server or have "manage server" permissions.

Commands

We have a wide rang of commands, following with the selected categories.

Moderation

  • Ban
    • Description : Let's you ban a targeted user.
    Usage : /ban < target > < optional: reason >
  • Kick
    • Description : Let's you kick a targeted user.
    Usage : /kick < target > < optional: reason >

    Utility

    • Help
      • Description : Supplies you with information about the bot, and supplies a list of commands.
      Usage : /help
    • NovaBot
      • Description : Supplies some important information about the bot.
      Usage : /novabot
    • Ticket Create
      • Description : Creates a ticket channel. So people can get some assistance.
      Usage : /ticketcreate

      Fun

      • Say
        • Description : Makes the bot say a custom message.
        Usage : /say < message >
      • Embed
        • Description : Makes the bot send a custom embed message.
        Usage : /embed < title > < description > < optional: footer > < optional: color >
      • Cute
        • Description : Call someone cute.
        Usage : /cute < user >
      • Love
        • Description : Tell someone you love them.
        Usage : /love < user >
      • Discord Mod
        • Description : Tell someone they are a discord moderator.
        Usage : /discordmod < user >
      • Dumb
        • Description : Tell someone that they are dumb.
        Usage : /dumb < user >
      • Father
        • Description : See who your real one is.
        Usage : /father
      • No Father
        • Description : See if you have one.
        Usage : /nofather
      • Rickroll Bird
        • Description : Look at the rickroll bird.
        Usage : /rickrollbird
      • Slap
        • Description : Slap someone, and make them cry.
        Usage : /slap < user >
